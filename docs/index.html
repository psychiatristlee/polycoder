Polymath

A cost-optimized, multi-model AI agent for your terminal, VS Code, and Chrome. It splits work into typed tasks, routes each to the cheapest model that's actually good at it, and tracks real cost by date + model.

VS Code

Chrome

Side-panel agent that automates the active tab: read, click, type, extract.

cd chrome-extension
npm i && npm run build
# Load unpacked

Why it's different

💸 Cheapest capable model per task

Each task maps to a skill (coding / reasoning / retrieval / speed). A cheap coder-tuned model handles edits; a reasoning model handles hard logic — picked by strength-per-dollar, not price alone.

📊 Real cost ledger

Every model call's actual tokens and cost are logged by date + model — SQLite in the CLI, chrome.storage in the browser. poly usage.

🔮 Recommend before you run

See cheapest / best-value / best-quality model combos and the estimated savings before spending a cent.

🔌 300+ models, one key

Reaches every major provider through OpenRouter. Guided key setup like Claude Code.

Install

Requires Node.js ≥ 22.5. Full docs in the README.

# CLI — from npm
npm install -g polycoder
poly login

# or from source
git clone https://github.com/psychiatristlee/polycoder.git
cd polycoder && npm install && npm link
